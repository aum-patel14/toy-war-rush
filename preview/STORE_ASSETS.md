# Store Assets Checklist — Toy War Rush

Use the browser demo (`OPEN_GAME.bat`) to capture Play Store materials.

## Required before Play Store submission

| Asset | Spec | How to create |
|-------|------|----------------|
| App icon | 512×512 PNG | Screenshot cannon + blue mob on desert lane; crop to circle in editor |
| Feature graphic | 1024×500 | Desert highway + “TOY WAR RUSH” title |
| Screenshots (6) | Phone 9:16 | Menu, gate choice, ×2 multiply, red mob wall, fortress clash, victory |
| Short description | ≤80 chars | Build your mob army and crush enemy fortresses! |
| Full description | ≤4000 chars | Mob Control-style runner: cannon stream, math gates, upgrades, 30 levels |
| Trailer | 30s MP4 | Follow `preview/RECORDING.md` |

## Output folder

Save exports to `docs/assets/`:

- `icon-512.png`
- `feature-1024x500.png`
- `screenshot-01.png` … `screenshot-06.png`
- `hero-trailer.mp4`

## Unity APK

Run `BUILD_ANDROID.bat` (Unity closed). Output: `Build/Android/ToyWarRush.apk`
